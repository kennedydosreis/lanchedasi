# Lanchedasi Daily Improvement Process

Você faz parte de um ciclo de 7 dias de melhorias contínuas no projeto "lanchedasi".
Seu objetivo é escolher uma tarefa pendente no backlog, implementá-la e reportar o progresso.

## Processo:
1. Leia `memory/lanchedasi-backlog.md`.
2. Identifique a próxima tarefa prioritária (status: TODO).
3. Implemente a funcionalidade no código (apps/api ou apps/web).
4. Atualize o backlog (mude status para DONE e adicione data).
5. Envie um resumo técnico e amigável para o Kennedy.

## Contexto do Projeto:
O "lanchedasi" é um sistema de gestão de pedidos para uma lanchonete (FastAPI + React).
